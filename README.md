# Chethan Gowda C — Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript. Features a black & white theme, animated particle network background, and smooth UI interactions.

## Live Demo

[View Portfolio](https://chethangowda25832-ctrl.github.io/chethanportfoilo)

## Features

- Animated polygon mesh particle network background
- Frosted glass card effects with backdrop blur
- Rotating profile image with scan line animation
- Typing animation for role titles
- Skills section with animated progress bars
- Projects grid with filter tabs
- Experience & Education timeline
- Certifications and Achievements
- Blog section
- Contact form with email integration
- Fully responsive — mobile, tablet, desktop
- Dark / Light theme toggle

## Tech Stack

- HTML5
- CSS3 (custom properties, backdrop-filter, animations)
- Vanilla JavaScript (Canvas API, IntersectionObserver)
- Font Awesome icons
- Google Fonts (Inter, JetBrains Mono)

## Project Structure

```
chethanportfoilo/
├── index.html       # Main portfolio page
├── cv.html          # CV / Resume page
├── style.css        # All styles
├── script.js        # All JavaScript logic
└── images/
    ├── pic.png      # Profile photo
    └── 3.1.jpg      # Project thumbnails
```

## Getting Started

No build tools needed. Just open `index.html` in a browser or serve with any static file server:

```bash
npx serve .
```
